# Yerba Mate Premium

Web ecommerce-ready inspirada en una marca DTC premium de yerba mate, construida con Next.js 15, TypeScript, Tailwind CSS, Framer Motion, Lucide React y React Hook Form.

## Scripts

```bash
npm run dev
npm run build
npm run start
```

## Estructura

- `src/app`: rutas principales.
- `src/components`: componentes por dominio.
- `src/data`: productos y tiendas estaticas.
- `src/lib`: utilidades compartidas.

## Logo

El logo actual es un wordmark editable en `src/components/layout/Header.tsx`. Reemplazalo por tu archivo real cuando lo tengas disponible.
